## 📌 Project Overview

An interactive AI-powered College Assistant Chatbot built using HTML, CSS, and JavaScript.  
This project helps students easily explore college-related information such as courses, fees, hostel facilities, placements, and more through a smart chat interface.

---

## 🚀 Features

- Interactive Chatbot UI  
- Voice Input (Speech Recognition)  
- Voice Output (Speech Synthesis)  
- Typing Animation & Smooth Scrolling  
- Predefined College Information Responses  
- Chat History Tracking  
- New Chat Reset Option  
- Responsive and Modern UI Design
